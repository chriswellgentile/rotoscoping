Multimedia &
Authoring (MMA)

An IT elective that bridges digital media production with programming — where creativity meets code.

Multimedia & Authoring is a core subject for IT students at Technological University of the Philippines — Taguig. It covers digital media production workflows including animation, video editing, 3D modeling, and interactive media authoring. This project serves as the final capstone output, combining rotoscoping techniques with 3D rendering.

🎭 What is Rotoscoping?

Rotoscoping is an animation technique where artists trace over live-action footage, frame by frame, to create realistic movement. Originally invented by Max Fleischer in 1915, it's now done digitally using tools like After Effects.

🪐 Connection to 3D

3D rendered elements from Blender are composited into the rotoscoped footage, creating a seamless blend of real-world and computer-generated imagery — a technique widely used in modern VFX pipelines.

🐍 Python Scripting

Blender's Python API (bpy) is used to automate the planet rendering pipeline — controlling materials, lighting, rotation keyframes, and camera paths programmatically for consistency across all 8 planets.

⏱️ 60-Second Output

Each week produces 5 seconds of rotoscoped footage. Across 12 weeks, this compounds into a 60-second final video showcasing the full progress and growth of the rotoscoping work.
